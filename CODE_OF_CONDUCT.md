# Codigo de Conduta

Este projeto segue uma postura simples: respeito, clareza e colaboracao.

## Comportamentos esperados

- Tratar pessoas com respeito.
- Explicar discordancias tecnicas com argumentos.
- Ajudar a manter issues e pull requests objetivos.
- Evitar linguagem ofensiva, ataques pessoais ou exposicao de dados privados.

## Aplicacao

Contribuicoes que violem esse codigo podem ser editadas, fechadas ou removidas pelos mantenedores.
